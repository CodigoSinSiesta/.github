# Codigo Sin Siesta

Comunidad tecnica para devs que quieren construir software de verdad: calidad sobre velocidad, comprension sobre atajos.

Aqui no vendemos humo. Aqui se viene a aprender, practicar y compartir ingenieria con criterio.

Sitio web: https://codigosinsiesta.github.io/

## Que hacemos

- Publicamos documentacion tecnica util y accionable para equipos reales.
- Compartimos contenido sobre arquitectura, herramientas y buenas practicas.
- Creamos presentaciones interactivas sobre IA aplicada con criterio de ingenieria.
- Montamos talleres hands-on para construir agentes de IA, MCP Servers y bases de conocimiento personales paso a paso.

  Presentacion interactiva (10 slides, 90-120 min) que acompana al starter del taller LLM Wiki. Astro + Svelte + Tailwind + GSAP. En vivo en https://codigosinsiesta.github.io/taller-llm-wiki-presentation/

- [taller-graphify-starter](https://github.com/CodigoSinSiesta/taller-graphify-starter)

- [taller-graphify-presentation](https://github.com/CodigoSinSiesta/taller-graphify-presentation)

## Enfoque

Sin atajos magicos. Sin tutorialitis.

Trabajamos con una base clara:

- Arquitectura entendible antes que complejidad accidental.
- Testing y verificacion como parte del flujo, no como extra.
- IA como herramienta de productividad, no como sustituto del criterio tecnico.
- Aprendizaje continuo con material real, practico y mantenible.

## Como participar

- Sigue la organizacion y revisa los repos fijados: https://github.com/CodigoSinSiesta
- Abre un issue con ideas, dudas, mejoras o propuestas de charla/taller.
- Lanza un PR pequeno y claro en documentacion o ejemplos.
- Comparte el material con tu equipo y trae feedback real para iterar.
